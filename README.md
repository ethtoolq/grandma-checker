# 🧓 Can My Grandma Use It?

Paste any URL — get a Grandma Score (0–100) based on how easy the site is for non-technical users. No AI, pure DOM parsing and UX heuristics.

→ **[www.grandmascore.site](https://www.grandmascore.site)**

## Stack

Node.js · Express · Cheerio · Vanilla JS · EN/RU

## Run locally

```bash
npm install
npm run dev
# http://localhost:3000
```

## How it works

Parses the page and penalizes for: too many buttons, deep navigation, missing CTA, technical jargon, wall of text.
